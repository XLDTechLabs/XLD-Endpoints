# Welcome to XLD Endpoints

XLD Endpoints is the library we use to:
- Authenticate via [XLoad](https://www.xload.io)
- Link accounts using Wallet Addresses similar to cryptocurrencies
- Reward XLD to users
- Convert your own Coins to XLD
